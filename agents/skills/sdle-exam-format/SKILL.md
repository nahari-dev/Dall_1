# SDLE Exam Format Reference

## Overview
The Saudi Dental Licensing Exam (SDLE) is administered by the Saudi Commission
for Health Specialties (SCFHS). This skill provides authoritative exam structure
data for generating accurate responses and mock exams.

## Instructions
1. When students ask about exam format, reference this data directly
2. When generating quiz questions, match the SDLE section distribution
3. When assessing readiness, map proficiency to SDLE section weights

## Exam Structure
- **Total questions**: 200 MCQs
- **Duration**: 4 hours (240 minutes)
- **Passing score**: 60% (some sources cite 70% for certain years)
- **Format**: Single-best-answer MCQs, 4 options each
- **Language**: English (some Arabic terminology accepted)

## Section Distribution
| Section | Weight | Topics |
|---------|--------|--------|
| Basic Sciences | 30% | Anatomy, Physiology, Biochemistry, Microbiology, Pathology |
| Clinical Sciences | 45% | Operative, Prosthodontics, Endodontics, Periodontics, Oral Surgery, Orthodontics, Pediatric Dentistry |
| Patient Management | 15% | Diagnosis, Treatment Planning, Emergency Management |
| Professional Ethics | 10% | Ethics, Jurisprudence, Infection Control, Radiology Safety |

## Registration
- Administered by SCFHS (Saudi Commission for Health Specialties)
- Eligibility: Graduates of accredited dental programs
- Registration via Mumaris+ portal
- Multiple testing windows per year
