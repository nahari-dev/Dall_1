# Dental Anatomy Reference

## Tooth Morphology
- Dental formula (permanent): 2-1-2-3 per quadrant = 32 teeth total
- Dental formula (primary): 2-1-0-2 per quadrant = 20 teeth total
- FDI notation: two-digit system (quadrant + tooth number)
- Palmer notation: quadrant symbol + tooth number

## Tooth Structure
- **Enamel**: 96% inorganic (hydroxyapatite), hardest substance in the body, ~2.5mm max thickness
- **Dentin**: 70% inorganic, 20% organic (Type I collagen), 10% water. Contains dentinal tubules
- **Pulp**: Connective tissue with blood vessels, nerves, odontoblasts. Provides vitality and sensation
- **Cementum**: 65% inorganic, covers root surface, attaches PDL fibers

## Periodontium
- **Gingiva**: Free gingiva, attached gingiva, interdental papilla
- **PDL**: 0.15-0.38mm wide, fiber groups: alveolar crest, horizontal, oblique, apical, interradicular
- **Cementum**: Acellular (cervical), cellular (apical)
- **Alveolar bone**: Cortical plates + cancellous bone, lamina dura visible on radiographs

## Biological Width
- Total: 2.04mm average
- Connective tissue attachment: 1.07mm
- Junctional epithelium: 0.97mm
- Violation → chronic inflammation and bone loss (Gargiulo et al., 1961)

## Instructions
1. When explaining dental concepts, reference specific anatomical structures
2. Use proper FDI notation for tooth identification
3. Relate anatomy to clinical significance for exam relevance
4. Include measurements and dimensions when clinically relevant
